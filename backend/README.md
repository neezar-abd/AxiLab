## ï¿½ Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Docker Services
```bash
# Di root folder
cd ..
docker-compose up -d
```

### 3. Setup Environment
File `.env` sudah tersedia dengan konfigurasi default untuk development.

### 4. Seed Database (Optional)
```bash
npm run seed
```

### 5. Start Server
```bash
npm run dev
```

Server: **http://localhost:5000**
